\begin{frame}{Normas y gobernanza técnica}
  \small
  \begin{itemize}
    \item \textbf{ISO/IEC/IEEE 12207:} orienta modularidad, ciclo de vida y separación de responsabilidades.
    \item \textbf{Bluetooth Core Specification:} sustenta interoperabilidad de la capa BLE.
    \item \textbf{RFC 8259 (JSON):} estructura el intercambio de datos entre la app y MuseRAG.
    \item \textbf{OWASP para LLM:} guía el tratamiento de entradas no confiables y el anclaje en fuentes.
    \item \textbf{ISO/IEC 25012:} refuerza calidad de datos y consistencia de metadatos.
    \item \textbf{CIDOC CRM, LIDO y Dublin Core:} inspiran estabilidad e interoperabilidad del contenido curatorial.
    \item \textbf{Ley N.\ 29733:} introduce criterios mínimos para el cuidado de datos del visitante.
  \end{itemize}
\end{frame}

\begin{frame}{Conclusiones}
  \small
  \begin{itemize}
    \item La tesis demuestra que existe espacio real para una solución de mediación cultural contextual en museos peruanos.
    \item La arquitectura propuesta integra BLE, smartphone BYOD, voz y RAG de manera técnicamente coherente.
    \item El prototipo supera el nivel conceptual: ya existe una base funcional de experimentación, integración y validación.
    \item La viabilidad económica es positiva en el escenario base y mejora cuando se consolida la base recurrente de museos activos.
    \item La propuesta queda alineada con criterios de interoperabilidad, calidad de datos, seguridad y evolución institucional.
  \end{itemize}
\end{frame}

\begin{frame}{Trabajos futuros}
  \small
  \begin{itemize}
    \item Ampliar la validación en más salas y con mayor cantidad de beacons.
    \item Incorporar pruebas controladas con visitantes y personal del museo.
    \item Expandir el corpus curatorial y evaluar estrategias adicionales de recuperación y control de alucinación.
    \item Migrar hacia una arquitectura institucional con mayor disponibilidad, observabilidad y seguridad.
    \item Profundizar interoperabilidad patrimonial y explorar expansión comercial hacia otros museos del clúster.
  \end{itemize}
\end{frame}
